# Portfolio profesional — Alejandro Dehesa Delgado

Portfolio técnico orientado a contratación junior. Está diseñado para transmitir un perfil serio: backend, producto, IA aplicada y cloud, usando proyectos como casos de estudio profesionales.

## Stack

- React
- Vite
- CSS moderno responsive
- Estructura de datos separada en `src/data/portfolio.js`

## Cómo ejecutarlo

```bash
npm install
npm run dev
```

Abrir:

```text
http://localhost:5173
```

## Cómo generar versión de producción

```bash
npm run build
npm run preview
```

## Qué debes actualizar antes de publicarlo

- Enlaces reales de GitHub, LinkedIn y CV.
- Demos reales de Aegis, SharedVault y BudgetWise.
- Repositorios reales de cada proyecto.
- Capturas reales dentro de cada caso de estudio.
- Métricas reales: endpoints, tests, deploy, coste AWS, performance o funcionalidades terminadas.

## Estructura principal

```text
src/
  App.jsx
  styles.css
  data/
    portfolio.js
  assets/
    alejandro-portfolio.webp
```

## Criterio del diseño

El portfolio prioriza:

- Claridad visual.
- Pocos proyectos, pero explicados con profundidad.
- Casos de estudio defendibles en entrevista.
- Stack agrupado por dominio.
- Texto profesional sin claims exagerados.
- Diseño sobrio tipo SaaS, sin animaciones innecesarias.
